## 🆘 IF SOMETHING DOESN'T WORK

**Check these first:**
1. Read the appropriate documentation file
2. Check `.env` file is correct
3. Verify backend is running
4. Clear browser cache (Ctrl+Shift+Delete)
5. Check browser console for errors (F12)
6. Review server logs

**Most common issues:**
- "Products not loading" → Check MongoDB URI
- "Admin login fails" → Restart server
- "404 on /api/*" → Check netlify.toml
- "Images not showing" → Update image paths
